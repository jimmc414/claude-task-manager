---
description: Mark a task complete (e.g., /done 1)
---

Mark a task as complete using the provided index.

Usage:
- `/done 1` → `tascli done 1`
- `/done 3 -c "finished early"` → `tascli done 3 -c "finished early"`

If no index is provided, first show the current task list so the user can pick one.

Run the tascli done command and confirm completion.
